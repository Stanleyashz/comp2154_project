# Project Overview

## Project Name

Job Application Tracker (JAT)

## Problem Statement

Job seekers often manage applications through scattered notes, email threads, spreadsheets, and bookmarks. This creates missed deadlines, incomplete follow-up, and poor visibility into application status.

## Proposed Solution

JAT is a web application that centralizes application tracking, authentication, application history, and status summaries in one place.

## Core Features

- User registration and login
- Create, view, update, and delete job applications
- Status-based tracking and summaries
- Notes and job links for each application
- Dashboard visibility for current progress
